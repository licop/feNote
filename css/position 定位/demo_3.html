Computed values are processed as far as possible without formatting the document. Some values, however, can only be determined when the document is being laid out. For example, if the width of an element is set to be a certain percentage of its containing block, the width cannot be determined until the width of the containing block has been determined.[1] The used value is the result of taking the computed value and resolving any remaining dependencies into an absolute value.
